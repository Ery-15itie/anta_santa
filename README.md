## サービス概要
Anta-Santaは、自己理解と相互支援が融合した、新しいコミュニティプラットフォームです。
一人ひとりが**Heartory Home(自分だけの家)**で心と向き合いながら、**Anta-Santa(どんな人でもサンタになれる温かい村)**で仲間と支え合う。プログラミング学習者を主ターゲットとして設計された、クリスマスが一年中続く居場所です。

## 🎄 Heartory Home（ハートリー・ホーム）とは

Heartory Home は、あなたの心の中にある“小さなサンタの家”をモチーフにした、自己理解のためのプライベート空間です。
ここでは、自分自身と静かに向き合う時間を持てます。

家の中の一つひとつの部屋には、心の状態・価値観・長所・夢・弱さ・思い出などあなたの“内側の世界”を整理するための仕掛けが詰まっています。

Heart（心）＋ Story（物語）＝ Heartory（ハートリー）

つまりここは、自身の “心の物語を育てていく家” です。

サンタのお家のように、あたたかく、やさしく、あなたを迎え入れる場所。
疲れた日も、嬉しかった日も、この家はいつでも変わらずあなたの味方というのがコンセプトです。

🌟 Heartory Home まとめ

Heartory Home は、あなたの
感情・思考・価値観・夢・弱さ・才能・思い出
を “部屋” という形で整理し、
「自分を知る」
「自分を肯定する」
「自分を育てる」
ための絵本のような自己理解空間とします。

Anta-Santa村では仲間と支え合い、
Heartory Homeでは自分と向き合う。

この2つの世界がつながることで、多くの人が挫折せず、成長し続けられる仕組みを作れるよう考えていきたいと思います。

### 開発者からのメッセージ(アプリに込めた想い)
子供の頃の私たちは毎年12月になると「今年は良い子だったかな？」と自分を振り返り、サンタさんからの評価を心待ちにしていました。あの頃のサンタさんは私たちの行いをすべて見ていて、公平に評価してくれる特別な存在でした。

でも大人になるとサンタさんはプレゼントを持って訪れてはくれません。じゃあ誰が私たちを評価してくれるでしょうか？上司でしょうか？それとも家族？友人？
実は、私たちの周りにいるすべての人が「サンタさん」なのです。

同僚があなたの仕事ぶりに感謝し、友人があなたの優しさに救われ、家族があなたの存在に安心する。私たちは誰かにとっての「サンタさん」であり、同時に誰かから贈り物のような評価をもらう存在なのです。そして、一番大切なのはあなた自身があなたの一番の自身の味方でありファンである。あなたがあなたの「サンタさん」であることです。

特にプログラミングなどの学習コミュニティでは、日々の学習努力や挫折を乗り越える姿勢、仲間との支え合いを評価していくことでコミュニティが成長すると考えています。しかし、GitHubの草や学習時間は見えても、そこに込められた苦労や成長・仲間への励ましは簡単に可視化できません。

「質問に丁寧に答えてくれて、学習のモチベーションが上がった」
「一緒に勉強していて、諦めずに続ける姿に刺激を受けた」
「エラーで困っていた時、親身になって話をきいてくれた」
「初心者の気持ちを理解して、共感してくれた」

こうした学習者の人間的な価値こそ、大人が求める「サンタさんからの評価」であり価値だと思います。

そして、もうひとり。大切なサンタさんがいます。

それは、あなた自身です。

誰よりもあなたの頑張りを知っているのは、あなた自身です。
Heartory Homeは、あなたが自分自身と向き合い、「よく頑張ったね」と自分にプレゼントを贈るための場所でもあります。

このシステムは、「みんなが誰かのサンタさんになれる場所、それよりも何よりも自身を抱きしめられる場所」を作ります。クリスマスが一年中続く、温かい居場所の実現を目指しています。 
GitHubの草や制作物からでは見えない各個人のソフトスキルや人柄が沢山の人の目に留まりますように！
